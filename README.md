## вңЁ Features

| Feature | Description |
|---|---|
| вһ• Add Transactions | Log income or expense with description, category, amount, date |
| рҹ“Ҡ Summary Cards | Live totals for Income, Expenses, and Balance |
| рҹҚ© Pie Chart | Spending breakdown by category (Chart.js doughnut) |
| рҹ“Ҳ Monthly Bar Chart | Income vs Expenses per month over time |
| рҹ”Қ Filter | Filter transaction list by All / Income / Expense |
| рҹ“Ҙ Export CSV | Download all transactions as a `.csv` file |
| рҹ—‘пёҸ Delete Entries | Remove individual transactions |
| рҹ’ҫ localStorage | All data persists across browser sessions |
| рҹ“ұ Responsive | Works on desktop and mobile |

## рҹҡҖ Setup & Run

### 1. Clone the repository
```bash
git clone https://github.com/YOUR_USERNAME/budgettracker.git
cd budgettracker
```

### 2. Open in browser
```bash
open index.html        # macOS
start index.html       # Windows
xdg-open index.html    # Linux
```

No API key needed. Works 100% offline.

## рҹ“Ҙ CSV Export Format

When you click **Export CSV**, a file like `budget_2025-03-22.csv` downloads with these columns:
